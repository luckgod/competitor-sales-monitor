## 阶段六：动态入口寻址 + Qwen2.5-VL 多模态引擎

### 6.1 开发任务

| 编号 | 文件 | 内容 |
|------|------|------|
| 6.1 | `src/ocr/entry_locator.py` | 动态入口寻址：PaddleOCR 扫描 `ENTRY_FEATURES` 特征词，提取坐标 |
| 6.2 | `src/ocr/vlm_extractor.py` | Qwen2.5-VL 多模态提取器：System Prompt 封装 + Ollama Vision API 调用 |
| 6.3 | `src/consumer/filter.py` | 升级骨架屏防御：SSIM 模板匹配 + Sobel 边缘梯度检查 |
| 6.4 | `src/pipeline/vlm_cache.py` | VLM 视觉状态机：增量差分裁剪 + KV-Cache slot 管理 |
| 6.5 | `src/core/gpu_guard.py` | 显存物理红线：OLLAMA_NUM_PARALLEL + MLOCK 前置校验 |
| 6.6 | `src/ocr/ocr_worker.py` | PaddleOCR 多进程隔离：multiprocessing.Queue 独立子进程 |

### 6.2 验收测试

| 编号 | 测试 | 判定红线 |
|------|------|---------|
| T6.1 | 含"超300人加购"的详情页截图 → OCR 命中 → 返回坐标 | 坐标偏差 < 5px |
| T6.2 | 不含特征词的详情页 → OCR 未命中 → 返回 None | 降级路径生效 |
| T6.3 | Qwen-VL API Mock → 返回标准 JSON → 解析为 Order 列表 | JSON Schema 校验通过 |
| T6.4 | 纯白卡片 → 熵 < 1.0 → 直接销毁 | 不调用 VLM |
| T6.5 | 骨架屏模板 → SSIM > 0.85 → 拦截 | Sobel variance 低于阈值 |
| T6.6 | 正常商品图 → SSIM < 0.5 → 通过 | 进入下游 |
| T6.7 | 连续 2 帧弹窗截图 → 第 2 帧仅传增量切片 | KV-Cache 命中，耗时 < 100ms |
| T6.8 | OCR 子进程 100 次调用 → 主线程帧率无下降 | GIL 隔离验证 |
| T6.9 | Ollama 显存 < 500MB 可用 → GPU Guard 报错退出 | 红字提示 |
